Projetos de Estruturas de Dados em Python
Este repositório contém três desafios desenvolvidos para a disciplina de Programação de Computadores.
Desafio 1: Sistema de Votação
Programa que registra votos para escolha de representantes de classe.
Funcionalidades:
* Entrada contínua de votos até digitar "fim"
* Armazenamento dos votos em lista
* Contagem de votos usando `count()`
* Verificação de vencedor ou empate
* Validação de candidatos

Desafio 2: Editor com Desfazer (Pilha)
Simulação de um editor de texto simples usando estrutura de pilha (LIFO).
Funcionalidades:
* Adicionar palavras com `append()`
* Desfazer última ação com `pop()`
* Exibir texto atual
* Tratamento de pilha vazia



#Desafio 3: Fila de Atendimento
Simulação de atendimento de alunos por ordem de chegada (FIFO).
Funcionalidades:
* Inserção de alunos com `append()`
* Atendimento com `pop(0)`
* Exibição da fila
* Tratamento de fila vazia

---
Conceitos Utilizados
* Listas em Python
* Estrutura de dados: Pilha (LIFO)
* Estrutura de dados: Fila (FIFO)
* Controle de fluxo
